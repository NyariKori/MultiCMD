# MultiCMD

[![English](https://img.shields.io/badge/Language-English-blue)](README.md)
[![Russian](https://img.shields.io/badge/Language-Русский-red)](README_RU.md)

MultiCMD is a powerful and lightweight plugin for Paper/Spigot/Bukkit servers that allows you to execute multiple commands in a single line using the `&` separator.

### Features
- **Chain Commands:** Combine several commands into one for efficient execution
- **Universal Compatibility:** Works with all vanilla commands and most commands from other plugins
- **Simple Syntax:** Easy-to-use structure—just separate commands with `&`
- **Powerful Automation:** Perfect for quick kits, teleportation sequences, and automated moderation

### Usage
`/test1 & ban player1 'Was behaving poorly'`

`/gamemode creative & fly on & heal & feed`

`/spawn & time set day & weather clear`

`/vip & eco give %player% 1000 & broadcast New VIP %player% joined!`

### Permissions
- `multicmd.use` - Permission to use the **MultiCMD** plugin
- `multicmd.reload` - Permission to use `/multicmd reload` command

- `/multicmd reload` - Reload plugin configuration
